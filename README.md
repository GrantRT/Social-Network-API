# Social-Network-API

## Introduction

An API for a social network web application where users can share their thoughts, react to friendsâ€™ thoughts, and create a friend list. Using Express.js for routing, a MongoDB database, and the Mongoose ODM.

## Technology Used

- Javascript
- Node.js
- MongoDB
- Mongoose
- Express

## Instructions

Firsty, make sure you have MongoDB installed on your computer.

Then, clone the repository and install dependencies by typing `npm i` into the console.

Then, start the server by typing `node server.js` into the console.

Then use insomnia to test the API.
